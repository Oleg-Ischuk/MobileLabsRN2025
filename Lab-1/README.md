# 📌 Запуск проекту

## 🔹 Клонування репозиторію

```sh
git clone <URL_РЕПОЗИТОРІЮ>
cd <ІМ'Я_ПАПКИ_ПРОЕКТУ>
```

## 🔹 Встановлення залежностей

```sh
npm install
```

## 🔹 Запуск проекту

```sh
npx expo start
```

## 🚀 Вибір способу відкриття додатка

```sh
# Веб-версія
w

# Емулятор Android
a

# Емулятор iOS (Mac + Xcode)
i
```

📱 Або скануйте QR-код у Expo Go (для реального пристрою).
